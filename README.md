# MVP mercado livre

Para o funcionamento do site nós precisamos de uma barra de pesquisa para buscar por um produto, um feed com os produtos, a página de visualização do produto e um botão de comprar agora

Obviamente é necessário que o usuário possa fazer cadastro para adicionar as informações pessoais
